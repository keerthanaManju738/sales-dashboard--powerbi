# Sales Insights Dashboard

## Project Overview
This project is an interactive sales dashboard created using Power BI to analyze revenue, sales trends, and regional performance.

## Tools Used
- Power BI
- MySQL
  

## Features
- KPI cards for revenue and sales
- Regional sales analysis
- Profit trend visualization
- Interactive filters and charts

## Project Files
- Dashboard PDF
- Dashboard Screenshots
  

## Outcome
The dashboard helps understand business performance and identify sales trends for better decision-making.
